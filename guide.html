How to install on your webserver?

1: Download the rest of the template

Your index.html file should now be downloaded. You must now download the rest of the template and extract it:

Download (zip file)

Extract this zip file to an empty folder, then replace the included index.html file with your custom one.

2: Upload to your webserver

This template was made to work well on any web hosting. It is simple static HTML, CSS and JavaScript. Simply upload the folder made in step 1 to your favorite host. You don't already have one? Here is a list of a few trustworthy hostings:

  • Cloudflare Pages
  • GitHub Pages
  • Artivain Hosting
